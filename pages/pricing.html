🔒 Security & Trust

Built with Security First

Your data, your privacy — always. Here's how we protect you.

🔑

License Key System

Unique encrypted license keys generated per user. Keys are validated locally — no server call needed after activation.

🛡️

Zero Data Upload

Audio recordings and transcripts NEVER leave your browser. Everything runs 100% client-side using Web Speech API.

⏳

Key Expiry Control

Each license key has an expiry date built into the hash. Expired keys are automatically rejected, no server needed.

🔒

IndexedDB Storage

Sessions are stored in your browser's secure IndexedDB — isolated per browser, never synced to any external server.

📧

Email Verification

License keys are tied to your email address. Sharing keys triggers detection and automatic key revocation.

🇮🇳

Razorpay Payments

Payments processed via Razorpay — India's most trusted payment gateway with PCI-DSS Level 1 compliance.
